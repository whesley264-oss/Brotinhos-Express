# Brotinho Premium

Este é um projeto de um site de uma página para a pizzaria Brotinho Premium, criado com HTML, CSS e JavaScript.

## Estrutura do Projeto

O projeto está estruturado da seguinte forma:

- `index.html`: O arquivo principal da aplicação, contendo toda a estrutura HTML do site.
- `css/`: Esta pasta contém os arquivos de estilo da aplicação.
  - `style.css`: O arquivo principal de estilos, contendo todo o CSS do site.
- `js/`: Esta pasta contém os arquivos de script da aplicação.
  - `script.js`: O arquivo principal de scripts, contendo todo o JavaScript do site.
- `img/`: Esta pasta contém as imagens utilizadas no site.

## Como usar

Para visualizar o site, basta abrir o arquivo `index.html` em seu navegador.
